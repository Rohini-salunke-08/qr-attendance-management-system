# QR Attendance Management System

## 📌 Description
A web application to automate attendance using QR codes.

## 🚀 Features
- QR code generation
- Attendance tracking
- Admin dashboard

## 🛠 Tech Stack
- Java, Spring Boot
- MySQL
- HTML, CSS, Bootstrap

## ▶️ How to Run
1. Clone the repository
2. Open in Spring Tool Suite
3. Run the application
